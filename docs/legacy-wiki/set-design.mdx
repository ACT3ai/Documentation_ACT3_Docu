## Overview

The **Set Design** feature in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") allows creators to generate, customize, and manage environments for their [scenes](/docs/legacy-wiki/scenes "Scenes") and [shots](/docs/legacy-wiki/shots "Shots").
Using AI-assisted world building, you can create anything from realistic locations to stylized fantasy backdrops, and integrate them directly into the [Editor workspace](/docs/legacy-wiki/editor "Editor").

## Key Capabilities

- **AI-Generated Environments:** Describe a setting in text (e.g., “Futuristic city at night with neon lights”) to generate 2D/3D backdrops.
- **Custom Assets:** Import props and models from external tools (e.g., Blender, FBX, USD).
- **Top-Down Planning:** Place props, [actors](/docs/legacy-wiki/digital-actors "Digital Actors"), and cameras using [Top-Down View](/docs/legacy-wiki/top-down-view "Top Down View").
- **Reusable Sets:** Save and reuse environments across projects for consistency.
- **Continuity:** Apply the same set across multiple scenes or episodes to maintain visual flow.
- **Lighting & Atmosphere:** Adjust day/night cycle, weather, or mood directly within the set.

## How to Use Set Design

1. Open the [Editor](/docs/legacy-wiki/editor "Editor") and switch to the **Top-Down View**.
2. Choose **Set Design → Create New Set**.
3. Provide a description or select from pre-built templates (house, street, forest, office, sci-fi).
4. Customize props, furniture, and world details.
5. Place [actors](/docs/legacy-wiki/digital-actors "Digital Actors"), [cameras](/docs/legacy-wiki/cameras "Cameras"), and [lighting](/docs/legacy-wiki/lighting "Lighting") in the space.
6. Save the set to your [asset library](/docs/legacy-wiki/asset-library "Asset library") for future use.

## Integration with Workflow

- [Storyboards](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") – preview how the set looks shot-by-shot.
- [Timeline](/docs/legacy-wiki/timeline "Timeline") – link set changes to scene transitions.
- [Generative Workflow Pipeline](/docs/legacy-wiki/generative-workflow-pipeline "Generative Workflow Pipeline") – integrates set generation with scripts and AI rendering.
- [Rendering System](/docs/legacy-wiki/rendering-system "Rendering System") – finalizes the environment in high resolution with Google Veo 3, Runway, or WAN AI.

## Example Use Cases

- Film set mockups for pre-visualization.
- Branded environments for [marketing videos](/docs/legacy-wiki/marketing-videos "Marketing Videos").
- Fantasy worlds for game trailers or animation.
- Reusable locations across episodic series.

## Best Practices

- Lock set layouts once approved to avoid continuity issues.
- Use [lighting presets](/docs/legacy-wiki/lighting "Lighting") for consistency across shots.
- Keep sets modular — reuse props and layouts to save time and credits.
- Always test camera placements in [Top-Down View](/docs/legacy-wiki/top-down-view "Top Down View") before final render.

## Troubleshooting

- **Set not rendering correctly** → Check asset compatibility or reduce complexity.
- **Props missing** → Re-import from [B-Roll/Asset Import](/docs/legacy-wiki/import-broll "Import BRoll").
- **Performance issues** → Use simplified proxy sets for previews.
- **Inconsistent look** → Apply a global [color grading LUT](/docs/legacy-wiki/color-grading "Color Grading").
