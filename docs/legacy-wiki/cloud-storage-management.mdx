## Overview

The **Cloud Storage Management** system in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") provides secure, scalable storage for all your projects, scripts, assets, and rendered outputs.
It allows creators and organizations to manage project files without worrying about local storage limits, ensuring seamless collaboration and fast access across devices.

## Key Capabilities

- **Centralized Storage:** All [projects](/docs/legacy-wiki/projects "Projects") are stored in the cloud for easy access.
- **Asset Management:** Upload and organize [B-Roll](/docs/legacy-wiki/import-broll "Import BRoll"), scripts, audio files, and custom assets.
- **Version Control:** Maintain versions of scripts, [scenes](/docs/legacy-wiki/scenes "Scenes"), and edits.
- **Access Control:** Share projects with your [team](/docs/legacy-wiki/organization "Organization") using role-based permissions.
- **Backup & Recovery:** Automatic backups with recovery options for deleted assets.
- **Scalable Capacity:** Expand storage as your project library grows.

## How to Use

1. Open the [dashboard](/docs/legacy-wiki/dashboard "Dashboard") or [Editor workspace](/docs/legacy-wiki/editor "Editor").
2. Navigate to **Cloud Storage** in the sidebar.
3. Upload new assets (scripts, audio, B-roll, textures, LUTs).
4. Organize assets into folders (by [project](/docs/legacy-wiki/projects "Projects"), [scene](/docs/legacy-wiki/scenes "Scenes"), or [shot](/docs/legacy-wiki/shots "Shots")).
5. Manage permissions for collaborators in the [Organization Settings](/docs/legacy-wiki/organization "Organization").
6. Download or export completed assets when needed.

## Storage Tiers

- **Basic Plan:** Limited storage bundled with standard [subscription plans](/docs/legacy-wiki/subscription-plan "Subscription Plan").
- **Professional/Studio Plans:** Larger storage quotas with shared [organization](/docs/legacy-wiki/organization "Organization") access.
- **Enterprise Solutions:** Custom storage capacity and dedicated servers.

## Integration with Workflow

- [Import Scripts](/docs/legacy-wiki/import-script "Import Script") and [Export Scripts](/docs/legacy-wiki/export-script "Export Script") are stored in your cloud workspace.
- [B-Roll assets](/docs/legacy-wiki/import-broll "Import BRoll") and footage can be organized into project folders.
- [Build Video](/docs/legacy-wiki/build-video "Build Video") outputs (renders, previews, exports) are automatically saved.
- Shared access allows multiple editors to collaborate on the same cloud-stored files.

## Best Practices

- Keep each project in its own folder for clarity.
- Use consistent naming conventions for [scenes](/docs/legacy-wiki/scenes "Scenes") and [shots](/docs/legacy-wiki/shots "Shots").
- Archive old projects to free active storage space.
- Regularly back up locally if working on high-stakes projects.

## Troubleshooting

- **Storage full** → Upgrade your [plan](/docs/legacy-wiki/subscription-plan "Subscription Plan") or purchase add-on storage.
- **Missing assets** → Check [permissions](/docs/legacy-wiki/organization "Organization") and activity logs.
- **Slow upload/download** → Verify internet connection; use compressed formats for large files.
- **Conflicting versions** → Use built-in version control to restore the correct file.

## Security

- End-to-end encryption for file transfers.
- Role-based access in organizations.
- GDPR and CCPA compliant data handling.
