## Overview

The **Color Grading** system in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") allows creators to adjust color, contrast, and tone across [scenes](/docs/legacy-wiki/scenes "Scenes"), [shots](/docs/legacy-wiki/shots "Shots"), or entire [projects](/docs/legacy-wiki/projects "Projects").
It works alongside [Style Presets](/docs/legacy-wiki/style-presets "Style Presets") and the [Rendering System](/docs/legacy-wiki/rendering-system "Rendering System") to ensure consistent visual mood and professional-quality output.

## Key Features

- **LUT Support:** Import and apply industry-standard LUTs (.cube).
- **Preset Styles:** Built-in looks for cinematic, natural, vintage, and high-contrast.
- **Custom Adjustments:** Fine-tune exposure, contrast, saturation, highlights, shadows, and temperature.
- **Shot Matching:** Match colors between multiple shots for continuity.
- **Global vs Local:** Apply grading per shot, per scene, or across entire projects.
- **Integration with Style Presets:** Use presets as a base, then refine with grading controls.

## How to Use

1. Open your [project](/docs/legacy-wiki/project "Project") in the [Editor Workspace](/docs/legacy-wiki/editor "Editor").
2. Switch to the **Color Grading** panel.
3. Choose a [style preset](/docs/legacy-wiki/style-presets "Style Presets") or import a LUT.
4. Adjust sliders for exposure, saturation, highlights, and shadows.
5. Apply globally or limit to a single [shot](https://wiki.act3ai.com/index.php?title=Shot&action=edit&redlink=1 "Shot (page does not exist)") or [scene](https://wiki.act3ai.com/index.php?title=Scene&action=edit&redlink=1 "Scene (page does not exist)").
6. Preview changes in the [Preview Pane](/docs/legacy-wiki/preview "Preview") before rendering.
7. Send to the [Render Queue](/docs/legacy-wiki/render-queue "Render Queue") for draft or final rendering.

## Best Practices

- Use LUTs for brand consistency across campaigns.
- Apply grading globally for a unified look, then adjust locally for key shots.
- Combine with [lighting](/docs/legacy-wiki/lighting "Lighting") adjustments in [Set Design](/docs/legacy-wiki/set-design "Set Design") for realism.
- Always preview at draft resolution before committing to 4K renders.
- Save custom looks to the [Asset Library](/docs/legacy-wiki/asset-library "Asset library") for reuse.

## Troubleshooting

- **Colors look washed out** → Check if LUT intensity is too high.
- **Inconsistent look between scenes** → Apply global grading or match shots manually.
- **Preset not applying** → Verify LUT file format (.cube supported).
- **Render doesn’t match preview** → Ensure “Include Color Grading” is enabled in [Export Settings](/docs/legacy-wiki/export-distribution "Export Distribution").
