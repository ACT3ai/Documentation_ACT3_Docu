## Overview

The **AI Task Assembly** in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") orchestrates the breakdown of creative work into discrete, automated tasks.
It ensures that scriptwriting, [digital acting](/docs/legacy-wiki/digital-actors "Digital Actors"), [camera movement](/docs/legacy-wiki/camera-movement "Camera Movement"), and rendering jobs are structured into a clear pipeline.
By automating task dependencies, AI Task Assembly enables faster iteration and smoother collaboration across the platform.

## Key Capabilities

- **Automated Breakdown:** Convert [script edits](/docs/legacy-wiki/script-editing "Script Editing") or [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard") outputs into granular AI tasks.
- **Dependency Management:** Link [Characters](/docs/legacy-wiki/characters "Characters"), [Scenes](/docs/legacy-wiki/scenes "Scenes"), [Shots](/docs/legacy-wiki/shots "Shots"), and [B-Roll](/docs/legacy-wiki/b-roll "B-roll") to ensure correct sequencing.
- **Parallel Execution:** Run multiple AI jobs (e.g., [Google Veo 3](/docs/legacy-wiki/google-veo-3 "Google Veo 3") render + [Runway Integration](/docs/legacy-wiki/runway "Runway")) in parallel.
- **Task Prioritization:** Optimize credit and compute usage with priority queues.
- **Collaboration Ready:** Teams can view task progress in the [Render Queue](/docs/legacy-wiki/render-queue "Render Queue") and assign overrides.

## How to Use

1. Inside the [Editor](/docs/legacy-wiki/editor "Editor"), select **Assemble Tasks** from the project menu.
2. Review automatically generated tasks tied to your script’s [scenes](/docs/legacy-wiki/scenes "Scenes") and [shots](/docs/legacy-wiki/shots "Shots").
3. Optionally adjust settings (task order, priority, credits).
4. Start execution: tasks are dispatched to the AI engines (e.g., [WAN AI](/docs/legacy-wiki/wan "Wan"), [Google Veo 3](/docs/legacy-wiki/google-veo-3 "Google Veo 3"), [Runway](/docs/legacy-wiki/runway "Runway")).
5. Monitor progress in the [Render Queue](/docs/legacy-wiki/render-queue "Render Queue") and review previews.

## Pricing & Credits

- Each task consumes credits according to its category:
  - Writing tasks: low credit cost.
  - Digital actor rendering: medium credit cost.
  - High-fidelity renders (Veo3 / Runway): higher credit cost.
- Bundling tasks in a sequence reduces redundant credit usage.

## Troubleshooting & Tips

- If tasks fail, check [credit balance](/docs/legacy-wiki/credit-system "Credit System") and network conditions.
- Use smaller test clips before triggering full production renders.
- Adjust dependencies so that dialogue and visual assets align correctly.
- For complex sequences, split task batches into multiple assemblies.

## Example Workflow

| Input | AI Task Assembly Output | Next Step |
| --- | --- | --- |
| Script draft in [Editor](/docs/legacy-wiki/editor "Editor") | Breakdown into beats, scenes, and shot tasks | Review in [Timeline & Storyboard](/docs/legacy-wiki/timeline-and-storyboard "Timeline & Storyboard") |
| Character notes | [Digital Actor](/docs/legacy-wiki/digital-actors "Digital Actors") tasks with performance prompts | Insert into scene assembly |
| Scene with action + camera notes | [Camera movement](/docs/legacy-wiki/camera-movement "Camera Movement") and [Lighting](/docs/legacy-wiki/lighting "Lighting") tasks | Send to [Render Queue](/docs/legacy-wiki/render-queue "Render Queue") |
| Final assembly | Batch rendering tasks | [Build Video](/docs/legacy-wiki/build-video "Build Video") export |
