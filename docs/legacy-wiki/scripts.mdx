## Overview

In [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI"), a **Script** is the narrative foundation of a [Project](/docs/legacy-wiki/project "Project").
Scripts define dialogue, stage directions, and scene structure, which are then expanded into [scenes](/docs/legacy-wiki/scenes "Scenes"), [shots](/docs/legacy-wiki/shots "Shots"), and final [videos](/docs/legacy-wiki/build-video "Build Video").
They can be written inside the [Script Editor](/docs/legacy-wiki/script-editor "Script Editor"), generated with [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard"), or imported/exported using standard formats.

## Script Workflow

1. **Create or Import:** Write directly in [Script Editing](/docs/legacy-wiki/script-editor "Script Editor") or use [Import Script](/docs/legacy-wiki/import-script "Import Script") from Final Draft (FDX), Fountain, or TXT.
2. **AI Expansion:** Use [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard") to expand one-liners into acts, beats, and dialogue.
3. **Editing & Collaboration:** Revise, comment, and lock sections with your team.
4. **Versioning:** Track changes with [Version Control](/docs/legacy-wiki/version-control "Version Control") and [Version History](/docs/legacy-wiki/version-history "Version History").
5. **Integration:** Link script lines to [Timeline](/docs/legacy-wiki/timeline-view "Timeline View") and [Storyboard Panels](/docs/legacy-wiki/storyboard-panels "Storyboard Panels").
6. **Export:** Share final drafts using [Export Script](/docs/legacy-wiki/export-script "Export Script") (FDX, Fountain, PDF).

## Features

- **Auto-Formatting:** Scene headings (INT./EXT.), character names, dialogue blocks.
- **Color Coding:** Distinguish scenes, shots, and revisions.
- **Annotations:** Inline comments for collaboration.
- **Side-by-Side View:** Compare original vs AI-edited drafts.
- **Script-to-Timeline:** Drag and drop directly into [Timeline View](/docs/legacy-wiki/timeline-view "Timeline View").

## Script Types

- **Original Drafts:** Written in-platform or imported.
- **AI-Assisted Drafts:** Expanded or rewritten via [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard").
- **Final Drafts:** Polished scripts ready for production or export.
- **Alternate Versions:** Branching scripts for testing different endings or pacing.

## Integration

- [Characters](/docs/legacy-wiki/characters "Characters") – assign lines and actions to actors.
- [Digital Actors](/docs/legacy-wiki/digital-actors "Digital Actors") – map dialogue to performances.
- [Set Design](/docs/legacy-wiki/set-design "Set Design") – link scene descriptions to locations.
- [Storyboards](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") – auto-generate shot panels from descriptions.
- [Timeline](/docs/legacy-wiki/timeline-view "Timeline View") – sequence script blocks into full projects.

## Best Practices

- Start with [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard") to generate structure.
- Keep dialogue concise for clarity and pacing.
- Use [versioning](/docs/legacy-wiki/version-control "Version Control") to manage alternate drafts.
- Sync scripts with [Timeline View](/docs/legacy-wiki/timeline-view "Timeline View") before rendering.
- Lock finalized scripts to prevent unwanted changes.

## Troubleshooting

- **Formatting incorrect** → Use [Import Script](/docs/legacy-wiki/import-script "Import Script") with Fountain or FDX for clean parsing.
- **AI edits missing** → Ensure AI suggestions are enabled.
- **Dialogue mismatched with actors** → Reassign in [Character Management](/docs/legacy-wiki/characters "Characters").
- **Export issues** → Try alternate format (FDX, Fountain, PDF).
