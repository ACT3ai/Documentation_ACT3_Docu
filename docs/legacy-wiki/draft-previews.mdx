## Overview

The **Draft Previews** feature in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") allows creators to quickly generate low-resolution, low-credit versions of their [scenes](/docs/legacy-wiki/scenes "Scenes") and [shots](/docs/legacy-wiki/shots "Shots").
Drafts help test pacing, camera angles, and dialogue timing in the [Editor Workspace](/docs/legacy-wiki/editor "Editor") before committing to high-quality, credit-intensive renders.

## Key Features

- **Fast Rendering:** Previews generate in seconds to minutes.
- **Low Credit Usage:** Saves significantly fewer [credits](/docs/legacy-wiki/credit-system "Credit System") than full 1080p or 4K renders.
- **In-Editor Playback:** Watch previews directly in the [Preview Pane](/docs/legacy-wiki/preview "Preview").
- **One-Click Updates:** Instantly re-render after script or shot changes.
- **Multi-Engine Support:** Available across [Google Veo 3](/docs/legacy-wiki/google-veo-3 "Google Veo 3"), [Runway Integration](/docs/legacy-wiki/runway "Runway"), and [WAN AI](/docs/legacy-wiki/wan "Wan").

## How to Use Draft Previews

1. Open a [Project](/docs/legacy-wiki/project "Project") in the [Editor Workspace](/docs/legacy-wiki/editor "Editor").
2. Select the scene, shot, or sequence you want to preview.
3. Click **Render → Draft Preview**.
4. Choose resolution (Draft, Low, Medium).
5. Preview appears in the [Preview Pane](/docs/legacy-wiki/preview "Preview").
6. If satisfied, upgrade to [Final Render](/docs/legacy-wiki/rendering-system "Rendering System").

## Credit Usage

| Resolution | Credits per 10s |
| --- | --- |
| Draft (low-res) | 1 credit |
| 720p | 2 credits |
| 1080p | 2–3 credits |
| 4K (final only) | 8–10 credits |

- Draft Previews are the cheapest option.
- Ideal for iteration before full rendering.

## Best Practices

- Always run draft previews before final renders to save credits.
- Use previews to test pacing, dialogue, and blocking.
- Re-render drafts after changes in [Script Editing](/docs/legacy-wiki/script-editor "Script Editor") or [Timeline View](/docs/legacy-wiki/timeline-view "Timeline View").
- Share draft previews with teammates for feedback before locking scenes.

## Troubleshooting

- **Draft won’t render** → Check [Render Queue](/docs/legacy-wiki/render-queue "Render Queue") and available credits.
- **Preview looks blurry** → Drafts are intentionally low-res; use final render for distribution.
- **Playback lagging** → Reduce preview resolution or simplify active layers.
- **Changes not updating** → Clear old draft and regenerate.

## Example Workflow

1. Draft script in [Script Editor](/docs/legacy-wiki/script-editor "Script Editor").
2. Block shots in [Top-Down View](/docs/legacy-wiki/top-down-view "Top Down View").
3. Generate Draft Preview to test pacing.
4. Refine scenes, dialogue, or camera placement.
5. Upgrade to final [Render](/docs/legacy-wiki/rendering-system "Rendering System") when satisfied.
