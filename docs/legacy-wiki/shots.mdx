## Overview

In [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI"), a **Shot** is the smallest production unit, representing a single continuous camera take within a [scene](/docs/legacy-wiki/scenes "Scenes").
Shots define exactly what the viewer sees at a given moment — including camera angle, framing, motion, and duration — and are the foundation for AI-assisted rendering in the [Editor Workspace](/docs/legacy-wiki/editor "Editor").

Shots are created from scene breakdowns and can be customized visually with the [Storyboard Panels](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") and [Top-Down View](/docs/legacy-wiki/top-down-view "Top Down View") tools.

## Key Features

- Precise control over camera position, lens, and movement
- Integration with AI rendering engines ([Google Veo 3](/docs/legacy-wiki/google-veo-3 "Google Veo 3"), [WAN AI](/docs/legacy-wiki/wan "Wan"), [Runway Integration](/docs/legacy-wiki/runway "Runway"))
- Shot-specific prompts for AI generation
- Real-time preview and regeneration
- Ability to re-order shots within a scene

## Creating a Shot

1. Open a [scene](/docs/legacy-wiki/scenes "Scenes") in the [Editor](/docs/legacy-wiki/editor "Editor").
2. Click **Add Shot** or split an existing shot.
3. Define:
   1. Camera type (e.g., wide, medium, close-up)
   2. Framing and composition
   3. Movement (pan, tilt, zoom, dolly)
   4. Duration in seconds
   5. AI prompt for visual generation
4. Attach a [storyboard panel](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") or design in [Top-Down View](/docs/legacy-wiki/top-down-view "Top Down View").

## Editing a Shot

- Select the shot in the Timeline or Scene view.
- Adjust technical parameters (lens, angle, camera motion).
- Modify AI prompt to refine visuals.
- Change timing by dragging shot edges in the Timeline.
- Regenerate the shot with updated parameters.

## Shot Types

| Shot Type | Description |
| --- | --- |
| Establishing | Sets scene context with location or environment view |
| Wide Shot | Captures entire subject and surroundings |
| Medium Shot | Frames subject from waist up for balance of detail and context |
| Close-Up | Focuses on subject detail, such as facial expression |
| POV Shot | Shows the scene from a character’s perspective |
| Tracking Shot | Follows moving subject through environment |
| Static Shot | No camera movement, fixed frame |

## AI Shot Workflow

- Write a short, descriptive AI prompt
- Select rendering engine ([Google Veo 3](/docs/legacy-wiki/google-veo-3 "Google Veo 3"), [Runway Integration](/docs/legacy-wiki/runway "Runway"), [WAN AI](/docs/legacy-wiki/wan "Wan"))
- Generate low-res preview
- Approve and send to [Render Queue](/docs/legacy-wiki/render-queue "Render Queue") for final render

## Collaboration Tools

- Shot-level comments for targeted feedback
- Shot locking to prevent unwanted changes
- Version history with rollback option

## Best Practices

- Keep prompts under 50 words for faster, more accurate AI rendering
- Match lens and camera movement to scene tone
- Use [storyboards](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") for visual consistency
- Preview before committing credits
