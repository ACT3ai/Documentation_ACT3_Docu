## Overview

The **Top Down View** in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") provides a bird’s-eye perspective of your scene.
It allows creators to visualize [Camera\_Angles](/docs/legacy-wiki/camera-angles "Camera Angles"), character placement, and spatial blocking from above, ensuring precise staging for [Shots](/docs/legacy-wiki/shots "Shots") and [Scenes](/docs/legacy-wiki/scenes "Scenes").

## 1. Purpose

- Plan camera positions relative to actors and environment
- Block out movement paths for [Characters](/docs/legacy-wiki/characters "Characters") and [Digital\_Actors](/docs/legacy-wiki/digital-actors "Digital Actors")
- Prevent continuity errors by referencing a spatial map
- Support quick adjustments during [storyboarding](/docs/legacy-wiki/storyboard-panels "Storyboard Panels")

## 2. Features

- **Scene Layout Grid:** View environment, props, and actor placement
- **Camera Icons:** Drag-and-drop [camera positions](/docs/legacy-wiki/camera-movement "Camera Movement") with orientation arrows
- **Character Tokens:** Represent actors with labeled markers for clarity
- **Multi-Camera Support:** Display overlapping shots for comparison
- **Path Tracing:** Visualize walking, running, or action sequences

## 3. How to Use

1. Open the [Editor](/docs/legacy-wiki/editor "Editor") and select **Top Down View** from the toolbar
2. Place characters on the layout using drag-and-drop tokens
3. Add cameras to define shot direction and focal length
4. Trace movement paths for [Scenes](/docs/legacy-wiki/scenes "Scenes") and [Shots](/docs/legacy-wiki/shots "Shots")
5. Sync adjustments with [Timeline\_&\_Storyboard](/docs/legacy-wiki/timeline-and-storyboard "Timeline & Storyboard") for preview

## 4. Benefits

- Quickly communicate staging to collaborators
- Combine with [Lighting](/docs/legacy-wiki/lighting "Lighting") and [Camera\_Movement](/docs/legacy-wiki/camera-movement "Camera Movement") for full scene planning
- Saves credits by reducing the number of trial renders
- Creates a reference map for consistent visual continuity

## 5. Roadmap

- Integration with virtual cinematography for AI-assisted blocking
- Export top-down diagrams as PNG for presentations
