# Comfy UI Integration

## Overview

The **Comfy UI Integration** in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") allows creators to access advanced, node-based AI video and image generation workflows directly within the platform.
Comfy UI is designed for power users who need fine-grained control over generative parameters, making it ideal for complex [scene creation](/docs/legacy-wiki/scenes "Scenes"), experimental [shot design](/docs/legacy-wiki/shots "Shots"), and integrating multiple AI models in a single render pipeline.

## Key Capabilities

- **Node-Based Workflow:** Build and edit generative pipelines visually using a drag-and-drop interface.
- **Multi-Model Integration:** Chain outputs from [Stable Diffusion XL](/docs/legacy-wiki/stable-diffusion-xl "Stable Diffusion XL"), [Flux](/docs/legacy-wiki/flux "Flux"), [Google Veo 3](/docs/legacy-wiki/google-veo-3 "Google Veo 3"), and other supported models.
- **Custom Parameters:** Set detailed control over resolution, frame rate, motion settings, lighting, and style weights.
- **Template System:** Save reusable workflow templates for different types of productions.
- **Batch Processing:** Render multiple shots or variations in parallel.

## How to Use

1. Open the [Editor workspace](/docs/legacy-wiki/editor "Editor") and select a [shot](/docs/legacy-wiki/shots "Shots") or [scene](/docs/legacy-wiki/scenes "Scenes") you want to process with Comfy UI.
2. Click **AI → Comfy UI** to launch the node editor.
3. Add and connect nodes for prompts, model selection, filters, and outputs.
4. Adjust parameters for each node as required.
5. Click **Generate** to produce your output and review in the preview panel].
6. Save your node setup as a workflow template if you plan to reuse it.

## Pricing & Credits

- Comfy UI renders are billed via the [Credit System](/docs/legacy-wiki/credit-system "Credit System") based on the models and resolution used.
- Multi-model workflows consume credits for each model in the chain.
- Example: [SDXL](/docs/legacy-wiki/stable-diffusion-xl "Stable Diffusion XL") → [Flux](/docs/legacy-wiki/flux "Flux") pipeline at 1080p may use 5 credits per 5 seconds.

## Tips & Best Practices

- Start with a prebuilt template to speed up workflow creation.
- Use [Set Design](/docs/legacy-wiki/set-design "Set Design") outputs as a base layer and enhance them with Comfy UI pipelines.
- Combine Comfy UI with [storyboard panels](/docs/legacy-wiki/timeline-and-storyboard "Timeline & Storyboard") to produce quick animatics.
- Keep track of credit usage when chaining multiple models in one render.

## Common Use Cases

- Advanced visual effects shots requiring multiple AI passes.
- Complex motion sequences that need layered rendering.
- Stylized experimental sequences for [social media](/docs/legacy-wiki/social-media-creators "Social Media Creators") or [film production](/docs/legacy-wiki/film-and-tv-professionals "Film and TV Professionals").
- Automated generation of multiple visual variations for client review.
