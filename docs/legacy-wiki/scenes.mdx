## Overview

In [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI"), a **Scene** is the primary storytelling unit within a [project](/docs/legacy-wiki/new-project "New Project"), containing a specific location, time, and set of [characters](/docs/legacy-wiki/characters "Characters").
Scenes organize your script into visual segments, which are then broken down into [shots](/docs/legacy-wiki/shots "Shots") for detailed production planning and AI-assisted rendering.

Scenes are managed directly in the [Editor Workspace](/docs/legacy-wiki/editor "Editor"), where you can write, edit, visualize, and preview them before finalizing.

## Key Features

- Structured breakdown of your [script](/docs/legacy-wiki/script-editor "Script Editor")
- Scene-level AI previews using [Google Veo 3](/docs/legacy-wiki/google-veo-3 "Google Veo 3"), [WAN AI](/docs/legacy-wiki/wan "Wan"), or [Runway Integration](/docs/legacy-wiki/runway "Runway")
- Integration with [storyboards](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") and [top-down set layouts](/docs/legacy-wiki/top-down-view "Top Down View")
- Scene locking for approved content in collaborative environments

## Creating a Scene

1. Open your project in the [Editor](/docs/legacy-wiki/editor "Editor").
2. Navigate to the **Script** or **Timeline** column.
3. Click **Add Scene** or drag a script block into the timeline to create a new scene.
4. Name the scene and define:
   1. Location
   2. Time of day
   3. Key characters
   4. Brief description
5. Save your changes — the new scene appears in both script and timeline views.

## Editing a Scene

- Double-click a scene in the Timeline or Script panel to open its editor.
- Modify dialogue, action lines, or descriptive text.
- Attach [storyboard panels](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") and [set layouts](/docs/legacy-wiki/top-down-view "Top Down View").
- Adjust [shot](/docs/legacy-wiki/shots "Shots") order and timing directly in the scene view.

## Scene Types

| Scene Type | Description |
| --- | --- |
| Standard | Full narrative sequence with location and dialogue |
| Montage | Multiple short moments combined into a sequence |
| Cutaway | Brief visual insert to add context or detail |
| Establishing | Location-introducing scene without main action |

## AI-Driven Scene Tools

- **AI Scene Expansion:** Suggests detailed action and dialogue based on scene outline
- **AI Tone Matching:** Adjusts scene style to match selected [story arc tone](/docs/legacy-wiki/ai-wizard "AI Wizard")
- **Automated Shot List:** Breaks down scene into recommended shots

## Collaboration Tools

- **Scene Locking:** Prevents accidental edits after approval
- **Comment Threads:** Leave feedback tied to specific lines or shots
- **Change History:** View past versions and revert as needed

## Best Practices

- Keep scene descriptions concise but visually rich
- Align scenes with [act structure](/docs/legacy-wiki/acts "Acts") for pacing and narrative flow
- Review AI-generated content before finalizing
