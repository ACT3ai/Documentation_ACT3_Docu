## Overview

A **Project** in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") is the main container for all creative work — including [scripts](/docs/legacy-wiki/scripts "Scripts"), [scenes](/docs/legacy-wiki/scenes "Scenes"), [shots](/docs/legacy-wiki/shots "Shots"), [digital actors](/docs/legacy-wiki/digital-actors "Digital Actors"), [sets](/docs/legacy-wiki/sets "Sets"), and final [video assemblies](/docs/legacy-wiki/build-video "Build Video").
Every production, whether a short social clip, a marketing video, or a feature-length film, begins as a Project.

## Project Structure

Projects in ACT 3 AI follow a nested hierarchy:

- **Project → Acts → Scenes → Shots → Assets (video, audio, B-roll, props, etc.)**

This structure ensures clear organization and easy collaboration at every stage of production.

## Creating a Project

1. From the dashboard, click **[New Project](/docs/legacy-wiki/new-project "New Project")**.
2. Enter a project name, description, and choose the type (Film, Ad, TikTok, Explainer).
3. Select aspect ratio (16:9, 9:16, 1:1) and estimated duration.
4. Save to begin editing in the [Editor Workspace](/docs/legacy-wiki/editor "Editor").

## Components of a Project

- [Script Editor](/docs/legacy-wiki/script-editor "Script Editor") – Write or import your screenplay.
- [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard") – Generate acts, beats, and narrative arcs.
- [Characters](/docs/legacy-wiki/characters "Characters") & [Digital Actors](/docs/legacy-wiki/digital-actors "Digital Actors") – Define story roles and cast them.
- [Set Design](/docs/legacy-wiki/set-design "Set Design") – Build or import environments for your scenes.
- [Timeline View](/docs/legacy-wiki/timeline-view "Timeline View") – Arrange and refine pacing.
- [Storyboards & Panels](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") – Visualize scenes shot by shot.
- [Build Video](/docs/legacy-wiki/build-video "Build Video") – Assemble final sequences.
- [Render Queue](/docs/legacy-wiki/render-queue "Render Queue") – Manage rendering jobs and outputs.

## Managing Projects

- **Folders:** Organize by campaign, client, or genre.
- **Versions:** Use [Version Control](/docs/legacy-wiki/version-control "Version Control") and [Version History](/docs/legacy-wiki/version-history "Version History") to track changes.
- **Collaboration:** Invite team members via [Organization Management](/docs/legacy-wiki/organization-management "Organization Management").
- **Locking:** Freeze approved [scenes](/docs/legacy-wiki/scenes "Scenes") or [shots](/docs/legacy-wiki/shots "Shots") to prevent edits.
- **Archiving:** Store inactive projects for long-term access.

## Billing & Credits

- Rendering and exporting consume [credits](/docs/legacy-wiki/credit-system "Credit System").
- Cost depends on resolution, length, and rendering engine (see [Billing Model](/docs/legacy-wiki/billing-model "Billing Model")).
- Credit usage can be monitored per project in the dashboard.

## Exporting Projects

Projects can be exported through [Export Distribution](/docs/legacy-wiki/export-distribution "Export Distribution") in multiple formats:

- **Video:** MP4, MOV, ProRes
- **Editing:** AAF, EDL, USD, FBX
- **Scripts:** [Export Script](/docs/legacy-wiki/export-script "Export Script") in FDX, Fountain, Markdown

## Best Practices

- Use descriptive names (e.g., “BrandX\_Campaign\_Fall2025”).
- Build draft previews before final renders.
- Reuse [characters](/docs/legacy-wiki/characters "Characters") and [sets](/docs/legacy-wiki/sets "Sets") across related projects for continuity.
- Monitor credit consumption with the [Credit Dashboard](/docs/legacy-wiki/credit-system "Credit System").

## Troubleshooting

- **Project won’t load** → Check [permissions](/docs/legacy-wiki/organization-management "Organization Management") or clear cache.
- **Missing assets** → Re-import via [Import Script](/docs/legacy-wiki/import-script "Import Script") or [Import B-Roll](/docs/legacy-wiki/import-broll "Import BRoll").
- **Render failed** → Verify [queue status](/docs/legacy-wiki/render-queue "Render Queue") and credit balance.
- **Out of credits** → Upgrade via [Subscription Plan](/docs/legacy-wiki/subscription-plan "Subscription Plan").

## Example Workflow

1. Create [New Project](/docs/legacy-wiki/new-project "New Project") → choose “Marketing Video.”
2. Generate story beats in [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard").
3. Draft script in [Script Editing](/docs/legacy-wiki/script-editor "Script Editor").
4. Add [actors](/docs/legacy-wiki/digital-actors "Digital Actors") and [sets](/docs/legacy-wiki/set-design "Set Design").
5. Assemble in [Timeline View](/docs/legacy-wiki/timeline-view "Timeline View").
6. Preview via [Render Queue](/docs/legacy-wiki/render-queue "Render Queue").
7. Final export through [Export Distribution](/docs/legacy-wiki/export-distribution "Export Distribution").
