## Overview

The **Import Script** feature lets you bring existing screenplays and outlines into [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") so you can immediately continue in the [Editor](/docs/legacy-wiki/editor "Editor") with structured [acts](/docs/legacy-wiki/acts "Acts"), [scenes](/docs/legacy-wiki/scenes "Scenes"), and [shots](/docs/legacy-wiki/shots "Shots").
Imports are parsed, auto‑formatted, and linked to downstream tools like [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard"), [Digital Actors](/docs/legacy-wiki/digital-actors "Digital Actors"), and the [Rendering System](/docs/legacy-wiki/rendering-system "Rendering System").

## Where to Find It

- From [Getting Started](/docs/legacy-wiki/getting-started "Getting Started") → **[New Project](/docs/legacy-wiki/new-project "New Project")** setup, choose **Import Script**.
- Inside the [Editor](/docs/legacy-wiki/editor "Editor"): **File → Import → Script** (adds to the current project).

## How Imports Are Structured

- Script elements are mapped to the ACT 3 hierarchy: **[Act](/docs/legacy-wiki/acts "Acts") → [Scene](/docs/legacy-wiki/scenes "Scenes") → [Shot](/docs/legacy-wiki/shots "Shots")**.
- **Scene Headings** (e.g., INT./EXT.) become [Scene](/docs/legacy-wiki/scenes "Scenes") cards.
- **Action & Dialogue** flow into the [Script Editor](/docs/legacy-wiki/script-editor "Script Editor") with speaker attribution.
- **Beat Markers** (from Fountain/Markdown) populate [Story Beats](/docs/legacy-wiki/story-beats "Story Beats").
- Optional page/line notes become annotations in the [Editor](/docs/legacy-wiki/editor "Editor").

## How to Use

1. Open **Import Script** from the project creation flow or the [Editor](/docs/legacy-wiki/editor "Editor").
2. Choose your file type: [FDX](/docs/legacy-wiki/final-draft "Final Draft") or TXT.
3. Select parsing options:
   1. Map # headings to [beats](/docs/legacy-wiki/story-beats "Story Beats") (Markdown/Fountain).
   2. Preserve original scene numbers.
4. Click **Import** to preview the parsed structure.
5. Confirm to create **Acts, Scenes, Shots** and open in the [Script Editor](/docs/legacy-wiki/script-editor "Script Editor").
6. (Optional) Run **[AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard")** to fill missing beats, logline, or theme.

## Formatting Hints (For Best Results)

- Use proper **Scene Headings** (e.g., `INT. LAB – NIGHT`) so the parser can segment reliably.
- Keep character names on their own lines in ALL CAPS for accurate dialogue mapping.
- In Markdown/Fountain, use `#` / `##` for beats and sub‑beats to seed [Story Beats](/docs/legacy-wiki/story-beats "Story Beats").
- Avoid embedding images in text imports—attach reference art separately to [Storyboard & Panels](/docs/legacy-wiki/storyboard-panels "Storyboard Panels").

## After Import: Next Steps

- Open the [Timeline](/docs/legacy-wiki/timeline "Timeline") to arrange scenes and add transitions.
- Cast roles in [Characters](/docs/legacy-wiki/characters "Characters") and assign [Digital Actors](/docs/legacy-wiki/digital-actors "Digital Actors").
- Block shots in [Top‑Down View](/docs/legacy-wiki/top-down-view "Top Down View") and add camera notes.
- Generate previews via [Google Veo 3](/docs/legacy-wiki/google-veo-3 "Google Veo 3"), [Runway Integration](/docs/legacy-wiki/runway "Runway"), or [WAN AI](/docs/legacy-wiki/wan "Wan").
- Finish and [Build Video](/docs/legacy-wiki/build-video "Build Video"); then use [Export Script](/docs/legacy-wiki/export-script "Export Script") / media export as needed.

## Troubleshooting

- **Missing scenes or wrong splits** → Check scene heading syntax; re‑run import with “Preserve scene numbers.”
- **Dialogue misattributed** → Ensure character names are uppercase and on their own lines.
- **Excessive paragraphs in one scene** → Add blank lines between action blocks for clearer segmentation.
- **Non‑English scripts** → Enable language detection to prep subtitles and dubbing tracks.
