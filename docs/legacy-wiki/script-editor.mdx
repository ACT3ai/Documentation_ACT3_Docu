## Overview

The **Script Editor** in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") is the primary workspace for writing, formatting, and editing [scripts](/docs/legacy-wiki/scripts "Scripts") within a [Project](/docs/legacy-wiki/project "Project").
It combines automatic screenplay formatting, AI-assisted writing, and collaboration features, making it easier to move from text to [scenes](/docs/legacy-wiki/scenes "Scenes"), [shots](/docs/legacy-wiki/shots "Shots"), and final [videos](/docs/legacy-wiki/build-video "Build Video").

## Key Features

- **Auto-Formatting:** Automatic handling of scene headings (INT./EXT.), character names, and dialogue blocks.
- **AI Assistance:** Expand ideas, rewrite dialogue, or generate alternates with [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard").
- **Color Coding:** Distinguish between acts, scenes, and shots.
- **Split View:** Compare original drafts with AI revisions side by side.
- **Annotations:** Add inline comments for collaboration.
- **Script-to-Timeline:** Drag and drop script blocks directly into [Timeline View](/docs/legacy-wiki/timeline-view "Timeline View").

## Workflow

1. Open the [Editor Workspace](/docs/legacy-wiki/editor "Editor") and select the Script Editor panel.
2. Draft your script directly or [import a script](/docs/legacy-wiki/import-script "Import Script") from Final Draft (FDX), Fountain, or Markdown.
3. Use [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard") to generate story arcs, beats, and scene ideas.
4. Assign [characters](/docs/legacy-wiki/characters "Characters") and [digital actors](/docs/legacy-wiki/digital-actors "Digital Actors") to dialogue lines.
5. Revise using AI suggestions or manual edits.
6. Lock completed sections with [Version Control](/docs/legacy-wiki/version-control "Version Control").
7. Push to [Timeline View](/docs/legacy-wiki/timeline-view "Timeline View") or [Storyboards](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") for visualization.

## Integration

- **[Scripts](/docs/legacy-wiki/scripts "Scripts")** – Central repository of dialogue and action.
- **[AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard")** – Generates beats and rewrites.
- **[Characters](/docs/legacy-wiki/characters "Characters") / [Digital Actors](/docs/legacy-wiki/digital-actors "Digital Actors")** – Assign roles directly from the script.
- **[Timeline View](/docs/legacy-wiki/timeline-view "Timeline View")** – Maps script blocks into time-based editing.
- **[Storyboards & Panels](/docs/legacy-wiki/storyboard-panels "Storyboard Panels")** – Generates panels from script descriptions.
- **[Export Script](/docs/legacy-wiki/export-script "Export Script")** – Save in FDX, Fountain, Markdown, or PDF.

## Collaboration

- Multiple team members can co-edit scripts in real-time.
- Inline comments support review workflows.
- Locked drafts ensure approved content isn’t overwritten.
- [Version History](/docs/legacy-wiki/version-history "Version History") tracks all revisions and contributors.

## Best Practices

- Use consistent formatting for clarity.
- Keep dialogue concise for pacing.
- Apply [Style Presets](/docs/legacy-wiki/style-presets "Style Presets") early for visual consistency in previews.
- Save manual [checkpoints](/docs/legacy-wiki/version-history "Version History") before major rewrites.
- Link [scenes](/docs/legacy-wiki/scenes "Scenes") to [Sets](/docs/legacy-wiki/set-design "Set Design") for faster visualization.

## Troubleshooting

- **Formatting incorrect** → Convert via [Import Script](/docs/legacy-wiki/import-script "Import Script") (Fountain/FDX preferred).
- **AI suggestions missing** → Ensure AI features are enabled in settings.
- **Export not aligned** → Re-export using [Export Script](/docs/legacy-wiki/export-script "Export Script") in the correct format.
- **Character mapping broken** → Reassign via [Character Management](/docs/legacy-wiki/characters "Characters").

## Example Workflow

- Step 1 [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard") generates a beat sheet.
- Step 2 Draft dialogue and action in Script Editor.
- Step 3 Assign [actors](/docs/legacy-wiki/characters "Characters") and link props.
- Step 4 Export via [Export Script](/docs/legacy-wiki/export-script "Export Script") for external review.
- Step 5 Lock version and send to [Timeline](/docs/legacy-wiki/timeline-view "Timeline View") for editing.
