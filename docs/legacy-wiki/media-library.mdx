## Overview

The **Media Library** in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") is the central repository for all project assets.
It allows you to import, organize, preview, and reuse content across multiple productions—ensuring creative consistency and efficient workflows.

## Key Capabilities

- **Unified Asset Hub:** Store scripts, video, audio, images, B-roll, and AI-generated clips.
- **Cross-Project Reuse:** Actors, props, and locations can be recalled across different projects to maintain visual continuity.
- **Search & Tagging:** Assets are indexed by type, project, tags, and usage rights.
- **Version Control:** Maintain multiple revisions of the same script, clip, or render.
- **Direct Workflow Integration:** Drag assets directly into the [Editor Timeline](/docs/legacy-wiki/editor "Editor"), [Build Video](/docs/legacy-wiki/build-video "Build Video"), or [Render Queue](/docs/legacy-wiki/render-queue "Render Queue").
- **B-Roll Imports:** Stock footage and supplemental clips are automatically cataloged for fast access.

## Supported Media Types

- **Scripts:** [Import Script](/docs/legacy-wiki/import-script "Import Script"), [Export Script](/docs/legacy-wiki/export-script "Export Script"), [Final Draft Integration](/docs/legacy-wiki/final-draft "Final Draft")
- **Video Clips:** AI-generated (from [Google Veo 3](/docs/legacy-wiki/google-veo-3 "Google Veo 3"), [Runway Integration](/docs/legacy-wiki/runway "Runway"), [WAN AI](/docs/legacy-wiki/wan "Wan"))
- **B-Roll:** Supplemental stock or reference footage (e.g., cityscapes, crowds)
- **Audio:** Voiceovers, soundtracks, effects
- **Images:** [Storyboard panels](/docs/legacy-wiki/storyboard-panels "Storyboard Panels"), references, AI stills

## Best Practices

- Name assets clearly (e.g., Scene01\_ShotA, Act2\_Music).
- Use folders for large projects to keep things organized.
- Delete unused previews to save storage and credits.
- Reuse tagged B-roll and audio for efficiency across projects.
