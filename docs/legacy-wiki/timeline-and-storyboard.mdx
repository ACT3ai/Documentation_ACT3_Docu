## Overview

The **Timeline & Storyboard** tools in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") allow creators to visualize, organize, and refine the structure of their [projects](/docs/legacy-wiki/projects "Projects").
They combine the linear sequencing of the [Timeline editor](/docs/legacy-wiki/timeline "Timeline") with the visual planning power of [Storyboards & Panels](/docs/legacy-wiki/storyboard-panels "Storyboard Panels"), bridging the gap between writing and rendering.

## Timeline

The [Timeline](/docs/legacy-wiki/timeline "Timeline") is where you arrange [scenes](/docs/legacy-wiki/scenes "Scenes"), [shots](/docs/legacy-wiki/shots "Shots"), and clips in time.
It provides precision editing, playback, and synchronization across [Digital Actors](/docs/legacy-wiki/digital-actors "Digital Actors"), audio, and visuals.

### Key Features

- Drag-and-drop [scenes](/docs/legacy-wiki/scenes "Scenes") and [shots](/docs/legacy-wiki/shots "Shots") directly from the [script editor](/docs/legacy-wiki/script-editor "Script Editor").
- Zoom for frame-level precision or wide-scene overview.
- Add transitions, overlays, and audio tracks.
- Lock or mute layers to simplify editing.
- Play back sections for quick review.

### Workflow

1. Import or draft your script in the [Script Editor](/docs/legacy-wiki/script-editor "Script Editor").
2. Generate [scenes](/docs/legacy-wiki/scenes "Scenes") and [shots](/docs/legacy-wiki/shots "Shots") using the [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard").
3. Drag script blocks onto the [Timeline editor](/docs/legacy-wiki/timeline "Timeline").
4. Adjust timing, pacing, and transitions.
5. Preview sequences in real-time.

## Storyboard

The [Storyboard](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") tool provides a visual breakdown of your script and timeline, showing each shot as an image or sketch panel.

### Key Features

- Auto-generate panels from [script descriptions](/docs/legacy-wiki/script-editor "Script Editor").
- Sketch or import [images and assets](/docs/legacy-wiki/import-broll "Import BRoll") for storyboards.
- Annotate with camera angles, notes, and directions.
- Sync storyboards with the [timeline](/docs/legacy-wiki/timeline "Timeline") for precise sequencing.
- Export as PDF or image deck for pitching or presentations.

### Workflow

1. Open the Storyboard panel from the [Editor workspace](/docs/legacy-wiki/editor "Editor").
2. Generate or sketch visual panels for each shot.
3. Align panels with corresponding [scenes](/docs/legacy-wiki/scenes "Scenes") in the [Timeline](/docs/legacy-wiki/timeline "Timeline").
4. Adjust order and refine notes for clarity.
5. Present or export as a storyboard deck.

## Integration

- Storyboards are linked to the [Timeline editor](/docs/legacy-wiki/timeline "Timeline") for scene synchronization.
- Both tools feed into the [Build Video](/docs/legacy-wiki/build-video "Build Video") process.
- Works alongside [Set Design](/docs/legacy-wiki/set-design "Set Design") and [Top-Down View](/docs/legacy-wiki/top-down-view "Top Down View") for visual consistency.
- Output is finalized via the [Rendering System](/docs/legacy-wiki/rendering-system "Rendering System") and [Render Queue](/docs/legacy-wiki/render-queue "Render Queue").

## Best Practices

- Always use Storyboards for early visualization before committing to renders.
- Use the Timeline for pacing, timing, and rhythm.
- Sync both tools to ensure scenes visually match script intent.
- Export a storyboard PDF for team or client review.

## Troubleshooting

- **Timeline won’t sync with script** → Refresh script import or check [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard") alignment.
- **Storyboard panels missing** → Re-generate from [script descriptions](/docs/legacy-wiki/script-editor "Script Editor") or re-import assets.
- **Playback lagging** → Lower preview resolution before full render.
