## Overview

The **Dashboard** in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") is the central hub where users can view project status, monitor AI usage, track storage consumption, and quickly access the most important tools. It provides an at-a-glance summary of active jobs, credit balance, and organizational resources.

## 1. Purpose

- Give creators, writers, and admins a single entry point
- Track [credit usage](/docs/legacy-wiki/credit-system "Credit System") and balances
- Visualize active and past render jobs in [Render\_Queue](/docs/legacy-wiki/render-queue "Render Queue")
- Display [storage usage](/docs/legacy-wiki/cloud-storage-management "Cloud Storage Management") per project/org
- Provide quick links to frequently used tools

## 2. Core Components

- **Project Overview:** List of active projects with status (drafting, rendering, completed)
- **Credit & Plan Summary:** Shows [plan tier](/docs/legacy-wiki/subscription-plan "Subscription Plan"), monthly credits, rollover bank
- **Storage Meter:** Visual indicator of cloud storage quota per org
- **Job Activity Feed:** Live updates from [Build\_Video](/docs/legacy-wiki/build-video "Build Video") and [AI\_Rendered\_Footage](/docs/legacy-wiki/ai-rendered-footage "AI Rendered Footage")
- **Quick Actions:** Start new [New\_Project](/docs/legacy-wiki/new-project "New Project"), open [Script\_Editor](/docs/legacy-wiki/script-editor "Script Editor"), or launch [AI\_Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard")

## 3. Metrics Displayed

- Current credits, rollover, and forecast usage
- Cloud storage usage by percentage and GB
- Job status: queued, rendering, finished, failed
- Recent [Scenes](/docs/legacy-wiki/scenes "Scenes") and [Shots](/docs/legacy-wiki/shots "Shots") generated
- Team activity (last edits, script changes, imports)

## 4. User Roles & Permissions

- **Writers:** See scripts, beats, and job previews
- **Directors:** Control [Storyboard\_Panels](/docs/legacy-wiki/storyboard-panels "Storyboard Panels"), [Camera\_Angles](/docs/legacy-wiki/camera-angles "Camera Angles") and rendering jobs
- **Admins:** Manage [Billing\_Model](/docs/legacy-wiki/billing-model "Billing Model"), [Credit\_System](/docs/legacy-wiki/credit-system "Credit System"), and [Organization](/docs/legacy-wiki/organization "Organization") settings
- **Viewers:** Read-only access to project summaries

## 5. Navigation

- Accessible from the main sidebar in [Editor](/docs/legacy-wiki/editor "Editor")
- Direct shortcuts to:

```
 * Import_Script / Export_Script  
 * Characters / Digital_Actors  
 * Storyboard_Panels / Top_Down_View  
 * Build_Video / Render_Queue  
 * Cloud_Storage_Management
```

## 6. Alerts & Notifications

- Low credit warnings at 20% and 5% thresholds
- Cloud storage alerts at 80% / 95% usage
- Job errors (e.g., failed render, invalid script import)
- Plan upgrade recommendations when resources are exceeded

## 7. Roadmap

- Personalized AI tips & suggestions per project
- Deeper analytics (per-user credit consumption, per-project storage cost)
- Activity timeline with undo/rollback options
- Team collaboration widget (chat + live notes)
