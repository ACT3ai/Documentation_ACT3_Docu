## Overview

The **Project Structure** in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") organizes all creative elements of your production — from story beats to rendered exports — into a unified workspace.
It helps users navigate between [Scenes](/docs/legacy-wiki/scenes "Scenes"), [Shots](/docs/legacy-wiki/shots "Shots"), [Characters](/docs/legacy-wiki/characters "Characters"), and assets while keeping billing, storage, and collaboration consistent.

## 1. Purpose

- Provide a clear hierarchy for film, video, and animation projects
- Enable smooth navigation between creative and technical layers
- Centralize assets (scripts, actors, renders) into one workspace
- Ensure [team collaboration](/docs/legacy-wiki/organization "Organization") with permissions and versioning

## 2. Core Levels

| Level | Description | Key Contents |
| --- | --- | --- |
| **Project** | The container for a film, campaign, or show | Title, genre, aspect ratio, [Subscription\_Plan](/docs/legacy-wiki/subscription-plan "Subscription Plan"), [Cloud\_Storage\_Management](/docs/legacy-wiki/cloud-storage-management "Cloud Storage Management") |
| **Acts & Beats** | Narrative structure created via [AI\_Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard") | Logline, theme, story arc, beats |
| **Scenes** | Individual moments in the story | Setting, [Lighting](/docs/legacy-wiki/lighting "Lighting"), [Top\_Down\_View](/docs/legacy-wiki/top-down-view "Top Down View"), [Storyboard\_Panels](/docs/legacy-wiki/storyboard-panels "Storyboard Panels") |
| **Shots** | Camera-based visual units | [Camera\_Angles](/docs/legacy-wiki/camera-angles "Camera Angles"), [Camera\_Movement](/docs/legacy-wiki/camera-movement "Camera Movement"), [Digital\_Actors](/docs/legacy-wiki/digital-actors "Digital Actors"), [Audio\_Tracks](/docs/legacy-wiki/audio-tracks "Audio Tracks") |
| **Renders** | Generated outputs | [AI\_Rendered\_Footage](/docs/legacy-wiki/ai-rendered-footage "AI Rendered Footage"), [Import\_BRoll](/docs/legacy-wiki/import-broll "Import BRoll"), previews, exports |

## 3. File & Asset Types

- Scripts → [Script\_Editor](/docs/legacy-wiki/script-editor "Script Editor"), [Import\_Script](/docs/legacy-wiki/import-script "Import Script"), [Export\_Script](/docs/legacy-wiki/export-script "Export Script"), [Final\_Draft](/docs/legacy-wiki/final-draft "Final Draft")
- Media → images, backgrounds, [Audio\_Tracks](/docs/legacy-wiki/audio-tracks "Audio Tracks"), narration
- Metadata → [Camera\_Instructions](/docs/legacy-wiki/camera-instructions "Camera Instructions"), timing
- Generated → [Google\_Veo\_3](/docs/legacy-wiki/google-veo-3 "Google Veo 3"), [Runway](/docs/legacy-wiki/runway "Runway"), [WAN\_AI\_Integration](/docs/legacy-wiki/wan "Wan") outputs

## 4. Navigation

- Use the **Dashboard** for global view of active projects
- Drill down into [Timeline\_&\_Storyboard](/docs/legacy-wiki/timeline-and-storyboard "Timeline & Storyboard") for beat-to-shot workflow
- Switch to [Top\_Down\_View](/docs/legacy-wiki/top-down-view "Top Down View") for staging and blocking
- Manage renders via [Render\_Queue](/docs/legacy-wiki/render-queue "Render Queue") and [Build\_Video](/docs/legacy-wiki/build-video "Build Video")

## 5. Collaboration

- Role-based access for [Member\_Management](/docs/legacy-wiki/member-management "Member Management")
- Shared [Documentation](/docs/legacy-wiki/documentation "Documentation") and notes
- Real-time [co-editing](/docs/legacy-wiki/collaboration "Collaboration") in the [Editor](/docs/legacy-wiki/editor "Editor")

## 6. Benefits

- Keeps large projects organized and scalable
- Bridges creative (writers, directors) with technical (VFX, editors) teams
- Links billing usage (via [Billing\_Model](/docs/legacy-wiki/billing-model "Billing Model") and [Credit\_System](/docs/legacy-wiki/credit-system "Credit System")) to actual creative work
- Supports future-proofing with [Cloud\_Storage\_Management](/docs/legacy-wiki/cloud-storage-management "Cloud Storage Management")
