## Overview

The **New Project** workflow in [ACT 3 AI](/docs/legacy-wiki/act3ai "ACT3AI") helps creators begin fresh productions with the right settings and structure.
Whether you’re building a short film, ad campaign, YouTube episode, or social media clip, every project starts here.

This module provides entry points into [Script Editing](/docs/legacy-wiki/script-editor "Script Editor"), [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard"), and [Timeline & Storyboard](/docs/legacy-wiki/timeline-and-storyboard "Timeline & Storyboard").

## Key Capabilities

- **Format Selection:** Choose content type (film, TV pilot, TikTok, ad, proof-of-concept).
- **Naming & Metadata:** Add project title, description, and tags.
- **Default Structure:** Pre-loads [Acts → Scenes → Shots](/docs/legacy-wiki/story-structure "Story Structure") templates.
- **Collaboration Setup:** Assign roles via [Member Management](/docs/legacy-wiki/member-management "Member Management").
- **Asset Linking:** Attach reference files, images, or previous drafts.

## How to Use

1. From the [Dashboard](/docs/legacy-wiki/dashboard "Dashboard"), click **New Project**.
2. Enter project name and optional description.
3. Select project type (short film, series episode, marketing ad, etc.).
4. Confirm template structure (Acts, Scenes, Shots).
5. Save and proceed to the [Editor workspace](/docs/legacy-wiki/editor "Editor").
6. Invite collaborators if needed.

## Example Workflow

| Step | Action | Result |
| --- | --- | --- |
| 1 | Start a New Project from the [Dashboard](/docs/legacy-wiki/dashboard "Dashboard") | Project shell created |
| 2 | Add title, description, and format | Metadata defined |
| 3 | Auto-generate structure via [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard") | Script outline drafted |
| 4 | Place [Digital Actors](/docs/legacy-wiki/digital-actors "Digital Actors") in [sets](/docs/legacy-wiki/2d-and-3d-sets "2D & 3D Sets") | Cast and environment ready |
| 5 | Preview shots in [Timeline & Storyboard](/docs/legacy-wiki/timeline-and-storyboard "Timeline & Storyboard") | Visual flow confirmed |
| 6 | Send to [Build Video](/docs/legacy-wiki/build-video "Build Video") | Final rendering begins |

## Tips

- Decide the format early—ads, short films, and long-form have different pacing.
- Use [AI Wizard](/docs/legacy-wiki/ai-wizard "AI Wizard") to jumpstart story arcs.
- Save multiple drafts under one project to track iterations.
- Organize assets early with [Media Library](/docs/legacy-wiki/media-library "Media Library").
