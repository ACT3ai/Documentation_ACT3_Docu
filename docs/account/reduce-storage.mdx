# Reduce Storage Usage

If you are approaching your storage limit or want to keep your organization clean, these steps reduce storage usage from most impactful to least.

To check your current usage: **Organization Settings → Storage**.

---

## 1. Delete Draft and Superseded Renders

Draft renders and earlier versions of regenerated shots accumulate quickly. These are the lowest-value files taking up the most space.

**Per shot:**
1. Open the shot in the editor
2. Click **Version History** on the shot card
3. Select versions you no longer need
4. Click **Delete Selected Versions**

**For an entire scene:**
1. Right-click the scene in the Outline
2. Select **Clean Up Renders → Delete Draft Versions**
3. Confirm — this removes all draft-quality renders for every shot in the scene, keeping only the highest-quality version of each

**For the whole project:**
1. Open **Project Settings → Storage**
2. Click **Delete All Draft Renders**
3. Confirm

Draft renders can be regenerated at any time (using credits) — they are not needed once you move to final production quality.

---

## 2. Delete the Videos in a Project (Keep the Project)

If a project is complete but you want to keep the script, structure, actor definitions, and settings without storing the rendered video:

1. Open **Project Settings → Storage**
2. Click **Delete All Rendered Video**
3. Confirm

This removes all generated video clips from the project but preserves everything else — the script, shot parameters, actor profiles, and set configurations. You can regenerate the video at any time if needed (at the standard credit cost).

Use this for completed projects in archive mode where you need the production record but not the video files.

---

## 3. Delete an Entire Project

If you no longer need a project at all:

1. Go to your Organization dashboard
2. Find the project in the project list
3. Click the three-dot menu on the project
4. Select **Delete Project**
5. Confirm deletion

Deleted projects move to **Trash**. No storage is freed until you empty the Trash.

**To empty Trash and free the storage:**
1. Go to **Organization Settings → Storage → Trash**
2. Click **Empty Trash**
3. Confirm — this is permanent and cannot be undone

---

## 4. Remove Unused Uploaded Assets

Uploaded files — reference images, B-roll footage, audio files, 3D models — that are no longer assigned to any project still consume storage.

1. Go to **Organization Settings → Asset Library**
2. Filter by **Unassigned** to see assets not linked to any active project
3. Select the assets you no longer need
4. Click **Delete**

You can also review uploaded assets per project: **Project Settings → Assets → View All**.

---

## 5. Archive Instead of Delete (Preserves Storage but Cleans Dashboard)

Archiving a project does **not** free storage — it only hides the project from the active project list. Use archiving to clean up the dashboard view; use deletion to actually reclaim space.

If you need to free storage, delete the project or its video content (see steps 2 and 3 above).

---

## 6. Upgrade Your Plan

If your production needs require more storage than your current plan provides:

1. Go to **Organization Settings → Billing**
2. Click **Upgrade Plan**
3. Choose a plan with the capacity you need

See [Storage](/account/storage) for storage limits by plan.

---

## Summary

| Action | Storage Freed | Risk |
|---|---|---|
| Delete draft renders | High — drafts are often 50%+ of storage | None — can regenerate |
| Delete all videos in project | High | Low — structure preserved |
| Delete entire project | Maximum | Medium — confirm you don't need it |
| Remove unused assets | Variable | Low |
| Archive project | None | — |
| Upgrade plan | N/A (adds capacity) | — |
