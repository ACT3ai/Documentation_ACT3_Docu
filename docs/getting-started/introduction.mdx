# Welcome to ACT3 AI

ACT3 AI is an AI-powered filmmaking platform that takes your script all the way to
production-ready video — without a camera crew.

## The Workflow

Every project in ACT3 AI follows the same cinematic pipeline:

```
Script  →  Beats  →  Scenes  →  Shots  →  Generate  →  Edit  →  Export
```

Each step maps directly to how professional filmmakers work:

| Step | What you do |
|------|-------------|
| **Script** | Paste or write your screenplay |
| **[Beats](/docs/concepts/beats)** | Mark each unit of dramatic action |
| **Scenes** | Group beats into continuous location/time sequences |
| **[Shots](/docs/concepts/shots)** | Define camera angle, movement, and framing |
| **Generate** | AI renders video for each shot using Veo 3, Runway, or ComfyUI |
| **Edit** | Arrange clips on the timeline |
| **Export** | Deliver to any platform or resolution |

## Who It's For

- **Filmmakers** — prototype sequences before production
- **Creators** — generate short-form video from scripts in minutes
- **Studios & agencies** — automate pre-viz and marketing content
- **Animators** — combine AI video with Blender 3D assets
