# Core Concepts

ACT3 AI is built on standard filmmaking vocabulary. Understanding these four concepts
unlocks everything else in the platform.

| Concept | Definition |
|---------|-----------|
| **Act** | A major story division — typically 3 acts per feature |
| **Scene** | A continuous sequence at one location and time |
| **Beat** | The smallest unit of action or emotional change in a script |
| **Shot** | A single camera angle or camera move within a scene |

See the individual pages for deeper detail on each concept.
