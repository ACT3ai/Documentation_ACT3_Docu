# Export

ACT3 AI supports exporting finished videos in professional formats for any destination — social media, streaming platforms, [video editors](/features/video-editor), or archive storage. Export settings are configured in the Build Video panel after your timeline is complete.

## Export Formats

### Video Formats

| Format | Best For |
|---|---|
| MP4 (H.264) | Web delivery, social media, general distribution |
| MOV (ProRes) | Professional editing workflows, post-production |
| WebM | Browser-optimized web delivery |
| AVI | Legacy format (use MP4 or MOV instead) |

**Recommendation:** Use MP4 for most distribution. Use ProRes when handing off to a [video editor](/features/video-editor) for further post-production work.

### Resolutions

- **720p** — Good for client previews and draft sharing
- **1080p** — Standard final delivery for most platforms
- **4K** — Premium output (requires Pro or Enterprise plan)

### Aspect Ratios

- **16:9** — Cinema, YouTube, TV
- **9:16** — TikTok, Instagram Stories, YouTube Shorts
- **1:1** — Instagram square posts
- **4:3** — Classic TV format

### Frame Rates

- **24fps** — Cinematic, film look
- **30fps** — Standard video, TV, web
- **60fps** — Smooth motion, gaming, sports content

### Script Export

Export your screenplay — including [Final Draft](/how-to/import-a-script) format — in these formats:

| Format | Use |
|---|---|
| Final Draft (FDX) | Professional writing workflow |
| Fountain / Markdown | Version control and sharing |
| PDF | Client review and distribution |
| Plain Text (TXT) | Simple sharing |

## Platform Presets

Use presets in the Export dialog to get format and resolution settings right automatically:

| Preset | Output | Use |
|---|---|---|
| YouTube | MP4 H.264, 16:9, 1920×1080 or 3840×2160 | YouTube, standard web |
| TikTok / Reels | MP4, 9:16, 1080×1920 | TikTok, Instagram Reels, YouTube Shorts |
| Instagram Square | MP4, 1:1, 1080×1080 | Instagram posts |
| Festival / DCP | ProRes 4444 master + optional DCP package | Film festivals, cinema delivery |
| Post-Production | ProRes 422, 16:9, project frame rate | Handoff to Premiere, Avid, DaVinci |

For TikTok and Reels exports, you can adjust the reframe position per shot before exporting to control which part of the 16:9 frame is kept in the 9:16 crop.

## Festival and DCP Export

Select the **Festival / DCP** preset. ACT3 AI renders a ProRes 4444 master and optionally packages a DCP (Digital Cinema Package) with the correct color space (P3-DCI), audio format (48kHz/24-bit), and subtitle track. Check your specific festival's technical requirements before submitting — requirements vary by festival.

## Multi-Platform Export

To produce versions for multiple platforms from the same project:

1. Set your primary aspect ratio to 16:9 for YouTube
2. Add additional export targets for TikTok (9:16) and Instagram (1:1)
3. ACT3 AI generates optimized exports for each format from the same project, adjusting reframe positions per shot for each target
4. All versions are rendered in a single queue batch

## How to Export

1. Complete your timeline assembly in Build Video mode
2. Click **Export** in the top toolbar
3. Choose a preset or configure format, resolution, and frame rate manually
4. Configure subtitles, language options, and delivery platform
5. Click **Export** — the job enters the [Render Queue](/features/render-queue)
6. Monitor progress in the Render Queue dashboard
7. Download the completed file when ready

## Distribution Options

**Direct Platform Upload** — Export directly to YouTube, TikTok, or private cloud storage with one click. Enter your platform credentials in [Organization Settings](/account/settings-organization) → Integrations.

**Secure Review Links** — Generate a secure preview link to share with clients or stakeholders. Links expire after a configurable time period.

**Investor and Pitch Deliverables** — Export annotated cuts and scene breakdowns alongside the video for presentations.

**Archive Masters** — Save ProRes master files for long-term storage and future re-editing.

## Subtitles and Localization

ACT3 AI can generate subtitles and multi-language dubs as part of the export process:

- **Auto-generated subtitles** — Based on your script and audio tracks
- **Accessibility compliance** — SRT and VTT subtitle file formats supported
- **Multi-language dubs** — Apply dubbed audio tracks to the same video for different markets
- **Lip sync adjustment** — Remap lip sync for dubbed dialogue automatically

Enable these options in the Export Settings panel before queuing your render.

## Credit Usage

Export consumes credits based on resolution and duration:

- Lower-resolution exports cost fewer credits
- ProRes exports at the same resolution cost the same as MP4
- Subtitle generation and audio dubs add a small additional cost
- Test exports in 720p or 1080p before committing to 4K to save credits

All credit estimates are shown before you confirm the export. See [credits](/account/credits) for how credit costs are calculated.

## Professional Workflow Formats

For post-production handoffs:

| Format | Use |
|---|---|
| EDL | Non-linear editor timelines (Premiere, Avid) |
| AAF | Advanced Authoring Format for professional editing |
| USD / FBX | 3D/VFX workflows |

These formats export the timeline structure and allow the receiving editor to reconnect to original source clips.

## Best Practices

- Run a 720p or 1080p test export before committing to 4K to verify color grading and audio sync
- Use platform-specific aspect ratios — 9:16 for TikTok, 16:9 for YouTube
- Generate subtitles for all videos for accessibility and SEO benefits
- Keep ProRes master files for any project you may need to revise or repurpose later
- Batch export multiple resolutions of the same content in a single queue job to save time

## Troubleshooting

**Export failed** — Check your credit balance and available storage space. Verify that all timeline tracks are enabled before exporting.

**Audio missing from export** — Confirm that audio tracks are enabled in the Build Video panel before queuing.

**Wrong aspect ratio** — Verify your project settings and timeline layout before exporting. The project aspect ratio set during creation determines the output ratio.

**Subtitles not syncing** — Re-export with subtitle generation enabled and verify that your audio track starts at timecode zero.

**4K export not available** — 4K requires a Pro or Enterprise subscription plan. Upgrade from Settings → Plans and Billing.
