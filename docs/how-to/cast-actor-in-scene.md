# How to Cast an Actor in a Scene

**Goal:** Connect a digital actor to a character name in your script so the AI uses that actor's appearance in every shot that character appears in.

---

## Steps

1. **Open the scene panel.** Navigate to the scene where you want to cast the actor.

2. **Go to the Characters tab.** In the scene panel, click the **Characters** tab. It shows the characters referenced in the script for this scene.

3. **Click the character name.** The character is shown as uncast if no actor is assigned. Click the assignment button next to their name.

4. **Select the digital actor.** A picker shows your available actors. Browse or search by name. Click the actor you want to cast in this role.

5. **Confirm the assignment.** The actor is now linked to that character name for this scene. All shots in the scene that feature this character will use this actor.

6. **Cast across the whole project (optional).** If this actor plays this character throughout the film, click **Apply to all scenes** to cast them project-wide at once.

7. **Verify appearance in a test render.** Generate a draft shot that features this character to confirm the actor's appearance renders correctly before committing to full quality renders.
