# How to Create a Digital Actor

**Goal:** Build a reusable digital actor with a defined appearance and voice so the AI renders them consistently across every scene they appear in.

A digital actor is created once and can be used in any project. Their look, voice, and motion capture data all travel with them.

---

## Steps

1. **Open the Characters directory.** In the left navigation, click **Characters**. The character library shows all actors available to your organization.

2. **Click + Add Character.** The character creation panel opens.

3. **Give the character a name.** Use the character name from your script. This is how they'll be linked to dialogue and scenes.

4. **Describe the character's appearance.** In the Appearance field, write a detailed physical description: age, gender, ethnicity, build, hair, distinguishing features. The more specific, the more consistent the AI renders them. Example: *"Woman, mid-40s, lean, dark shoulder-length hair with grey at the temples, sharp cheekbones, wears reading glasses."*

5. **Upload a reference photo (optional but recommended).** If you have a photo of a real person, an actor, or a character illustration that captures the look you want, upload it. The AI uses it alongside the text description.

6. **Define the base wardrobe.** Describe what the character wears by default. This is their costume unless a scene overrides it. See [How to define a character's costume](/how-to/define-character-costume).

7. **Assign a voice.** Go to the Voice tab. Choose from the TTS voice library — select by gender, age range, accent, and tone. Play samples to find the right match.

8. **Save the character.** Click **Save**. The actor is now available to cast in any scene.
