## SECTION 1: High-Level Summary

### Product Mission
**"Write less, more creative control, far less labor."**

ACT 3 AI transforms rough story ideas or finished scripts into production-ready, AI-generated movies or video—complete with scenes, cinematography, lighting, and 3D set control. The platform enables creators to visualize faster with extensive automation while spending budgets on creativity instead of logistics.

### Platform Type
ACT 3 AI is a hosted internet WebApp product designed as a Software-as-a-Service (SaaS) monthly subscription platform with metered usage based on consumption.

### Core Users

**Primary Users:**
- People creating Movies
- TV show producers
- Social media video creators
- Marketing video professionals
- Focus on storytelling-driven video content

**Secondary Users (Blender Experts):**
- VFX experts who create 3D Sets
- Often create sets rather than complete movies
- Appreciate the unique integration of Blender with Generative AI video

### The Product: Five Core Capabilities

**1. Script Import**
- Full scripts can be imported directly into the platform
- Supports various formats for professional screenplay integration

**2. AI Story Expansion**
- For creators with high-level story ideas but no complete script
- AI system expands ideas into full scripts
- Creates narrative depth and structure automatically

**3. Generative AI Video Integration**
- Leverages cutting-edge "Generative AI Video" technology where LLM AI creates videos
- Uses lower-level services but shapes the creative vision
- Controls "what" will be visualized in generated videos

**4. Comprehensive Structure Management**
- Structures content up to 2-hour movies or TV shows
- Manages Beats, Scenes, Shots, and cinematography details
- Provides professional-grade organizational framework

**5. Extensive Visual Control**
- Location scouting
- Set design
- Cinematography
- Lighting design
- Directorial control over all visuals
- Functions performed within AI-generated video creation

### Technology Stack

The platform integrates with leading generative AI video technologies:
- **Flux**
- **Stable Diffusion SDXL**
- **Runway AI**
- **Google Veo 3**
- **ComfyUI**
- **Hunyuan**
- **Wan 2.1**

### 2.0 Core Billing Model: Credits

**Credit System Foundation:**

**Definition:**
Credits are internal currency for resource-intensive actions

**Key Principles:**

1. **Consumption-Based**
   - AI script generation
   - Video shot creation
   - 3D set rendering
   - Text-to-speech conversion

2. **Transparency**
   - Cost clearly displayed before action
   - No hidden charges
   - Upfront credit requirements

3. **Tiered Allocation**
   - Different monthly credit amounts per plan
   - Scales with subscription level
   - Clear upgrade path

**Metered Actions:**
- Each task consumes specific credit amount
- Real-time balance tracking
- Overage options available

### 3.0 The Organization: Billing Heart

**Organization Model:**
Similar to GitHub or Figma workspace structure

**Definition:**
- Isolated workspace owning:
  - Projects
  - Members
  - Subscription plan
  - Credit pool
  - Payment method

**Automatic Creation:**
- New user signup creates default Organization
- Example: "Jane Doe's Personal Projects"

**Multiple Organizations:**
- Single user can create/join multiple Organizations
- Separate personal from professional work
- Example: "Jane's Personal Projects" vs "Big Movie Studio Inc."

**Ownership:**
- Organization legally owns:
  - All projects
  - Created content
  - Generated assets

### 4.0 User and Project Structure

**Hierarchy:**

```
User (Individual Account)
└── Organization (Workspace)
    ├── Subscription Plan
    ├── Credit Pool
    ├── Payment Method
    └── Projects
        ├── Project 1
        ├── Project 2
        └── Project N
```

**Key Relationships:**
- User can belong to multiple Organizations
- Organization owns multiple Projects
- Project belongs to ONE Organization
- Credits drawn from parent Organization pool
